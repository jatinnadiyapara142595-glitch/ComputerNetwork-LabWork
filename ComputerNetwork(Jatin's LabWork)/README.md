# 🔐 Computer Networks & Cryptography Lab (MCA)

This repository contains practical implementations of classical cryptographic algorithms and basic socket programming using Python.

---

## 📚 Contents

### 🔸 Unit 5 – Cryptography Programs

1. **XOR Cipher**
   - Performs XOR operation on characters
   - Demonstrates basic encryption concept

2. **Caesar Cipher**
   - Shift-based substitution cipher
   - Supports encryption and decryption

3. **Hill Cipher (2x2)**
   - Matrix-based encryption technique
   - Uses modular arithmetic

4. **Playfair Cipher**
   - Digraph substitution cipher
   - Uses 5x5 matrix

5. **Rail Fence Cipher**
   - Transposition cipher using zigzag pattern

### 🌐 Socket Programming (Computer Networks)

#### 🔹 TCP Communication
- **Server (`serverchat1.py`)**
  - Listens for incoming client connections
  - Sends and receives messages

- **Client (`clientchat1.py`)**
  - Connects to server
  - Enables two-way communication

---

#### 🔹 UDP Communication
- **UDP Server (`udp_server.py`)**
  - Receives datagrams
  - Sends response to client

- **UDP Client (`udp_client.py`)**
  - Sends message to server
  - Receives server response
